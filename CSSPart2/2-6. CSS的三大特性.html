(1) 层叠性

给同一个选择器的同一个样式设置不同的值时，后面的会覆盖掉前面的，例如: 
        div {
            color: red
        }

        div {
            color: pink
        }
        有两个同时修饰div-color，只有下面这个会生效，在debug调试工具中，Style这一栏可以清晰的看到哪些样式正在生效

(2) 继承性

子标签会继承父标签的一些样式(主要是和文字，颜色相关的，如text-, font-, line-， color等)，例如： 
        div {
            color: red;
            font-size: 2em;
        }
        同样的可以在Debug-Style中看到这个样式是从哪里继承来的
        
        // 有一个特殊的需要注意一下：行高的继承，当父元素设置行高没带单位时，子元素的行高会根据当前文字尺寸来计算，如果子元素没有设置文字尺寸，则会采用父元素的尺寸例如：
        <div>
            <p class="p1" style="font-size: 20px;">123</p>
            <p class="p2" style="font-size: 30px;">456</p>
        </div>
        此时 p1的行高就应该是20*1.5 = 30px；p2的行高应该是30*1.5 = 45px
