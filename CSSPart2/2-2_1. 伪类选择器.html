伪类选择器

伪类选择器用于筛选出特定状态下的标签，语法为 标签名:关键字，例如： 
        p:hover {
            color: red;
        }

        p:active {
            font-size: 2em;
        }

伪类选择器可以选择的标签有很多，常见的有：链接、结构、表单等

（1）链接伪类选择器

        // 伪类选择器也可以和其他的选择器组合使用
        .fakeClassFilter a:link {
            color: green;
        }

疑问：

【Q1】CSS样式在style中的顺序对绘制有什么关系？（当a:hover写在a:visited之前时，对于已经点过的链接hover效果不生效，调换顺序之后表现正常）
 【A】对于链接伪类选择器来说，请按照固定顺序来写： link -> visited -> hover -> active

【Q2】为什么给body设置了CSS样式之后，body中链接的样式并没有生效？
 【A】由于a(链接)在浏览器中有默认的样式，所以需要给链接单独指定样式

（2）焦点伪类选择器

:focus伪类选择器主要用来筛选获得焦点的表单元素；焦点可以理解为光标，通常情况下<input>标签这一类的表单元素才能获取较点，故而这个选择器主要针对表单元素。 
        //同样 它也可以作为一个基础选择器与其他的选择器进行复合筛选操作
        .formFocusFilter input:focus {
            background-color:red;
        }
