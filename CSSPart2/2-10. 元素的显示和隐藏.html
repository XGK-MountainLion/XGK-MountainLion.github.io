元素的显示和隐藏

(1) display

display: block; // 除了可以将元素设置为块级元素，也有显示的功能
display: none; // 隐藏(实际开发中用的更多一些)

(2) visibility

visibility: visible; // 显示
visibility: hidden; // 隐藏

(3) overflow

这个属于用来标识当元素超出盒子边界时该如何处理
overflow: visible; //显示
overflow: hidden; //隐藏
overflow: scroll; //横向和纵向的滑动条，注意无论元素是否超出边界都会有这个滑动条出现
overflow: auto; //当元素超出边界时才会有滑动条出现
需要注意的是：如果盒子使用了position，且他的子元素位置有超出盒子边界的情况存在时，要慎用hidden，因为超出的部分会被隐藏掉

display和visibility在隐藏上的区别在于：display更像是将active设置为false，后面的标准流会补上占位，而visibility是将alpha值设为了0，只是看不见，但依然占据当前位置
