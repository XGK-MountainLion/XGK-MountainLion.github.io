font属性

font属性用于设置字体、大小、粗细、文本样式（斜体）
font属性一般用来修饰Body标签 
        p {
            // 文字样式：normal, italic(倾斜)，常用于修饰<em> 标签，将倾斜的字体恢复正常
            font-style: italic；

            // 字体粗细: normal(默认，等同于400). bold(等同于700)， bolder(特粗), lighter(细体);实际开发中一般都填数字 
            font-weight: bold;

            // 设置大小，单位px是像素的意思；建议不要采用浏览器默认的size，不同的浏览器可能标准不一样，通常手动设置为16或14（一般网页的size） 
            font-size: 20px;

            // 设置字体, 支持设置多字体，每个字体之间使用逗号分隔；多字体的含义是从第一个开始遍历，直到遇到用户已安装的字体 
            font-family: "Microsoft Yahei", Arial;
        }
这几个声明也可以写成一行，以空格划分，但顺序必须固定 
        p {
            // 固定顺序为：style, weight, size, font-family；size和family为必填项，style和weight如果不填则取默认值 
            font: normal 700 20px "Microsoft Yahei";
        }
