CSS的引入方式

        // 行内: 直接写在标签内部，各个声明之间由分号分隔; 只能作用于当前这个标签【不可复用】
        <p style="color: red;"> 段落文本</p style=""> 段落文本

        // 内部：在head标签中添加style标签，只能作用于当前HTML文件【局部复用】
        <style>
            p {
                color: red;
            }
        </style>

        // 外部：写在CSS文件中，能作用于所有引入了这个CSS文件的HTML; 在CSS文件中，不需要添加style标签了，专注于内容即可【可复用】
        p {
            color: red;
        }
        
        // 在HTML的head标签中，使用link标签来引入CSS文件, 例如：
        <link rel="stylesheet" href="../一、Demo_CSS.css">
